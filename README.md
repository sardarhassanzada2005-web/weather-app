### Overview
    A simple weather application that allows users to check current weather
    information for a selected location using live data from a weather API.
###  Features
    - Search weather by city name
    - Display temperature and weather condition
    - Handle basic API errors
    - Country flag
### Technologies
    - HTML
    - CSS
    - JavaScript
    - Public Weather API
### What I Learned
    - How to fetch data from an external API
    - How to use async/await in JavaScript
    - How to handle user input
    - How to work with JSON data
    - How to handle simple errors form API
### Purpose
    I built this project to practice using APIs and working with real
    weather data.
